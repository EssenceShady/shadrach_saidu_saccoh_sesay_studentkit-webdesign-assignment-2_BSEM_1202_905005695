# Student Kit

## Digital Educational Platform for Sierra Leone Students

Student Kit is a responsive educational web platform designed to help students in Sierra Leone access examination resources such as past question papers, educational updates, quizzes, and result-checking support.

The platform mainly supports students preparing for:

- NPSE
- BECE
- WASSCE
- IGCSE



# Problem Statement

Many students in Sierra Leone struggle to access reliable and organized examination materials digitally. Most educational resources are distributed physically, making them difficult to access, especially for students in remote areas.

Student Kit was created to provide a centralized digital solution for examination preparation and educational support.



# Features

- Past Question Papers
- Educational News & Updates
- Subject Categories
- Result Checker Support
- Responsive Design
- Mobile Friendly Interface
- Contact Section
